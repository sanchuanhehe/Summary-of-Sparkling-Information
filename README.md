## 主要内容

- **TXS-00001-2025**: 星闪无线通信系统架构 - 定义星闪系统的整体架构

### 软件资料

包含星闪相关软件资源链接及常见问题解答。

### 硬件文档

包含星闪芯片和开发板的硬件设计指南、规格说明书及原理图。

## 关于星闪

星闪(SparkLink)是一种新一代无线短距通信技术，具备低时延、高可靠、低功耗等特性，广泛应用于智能汽车、智能终端等领域。
